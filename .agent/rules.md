# Reglas del Usuario

1. **Idioma de Respuesta**: Siempre contestar en **Español (Castellano)**, independientemente del idioma del input, a menos que se solicite explícitamente lo contrario.
   - Esto aplica a explicaciones, chats y resúmenes.
   - El código debe seguir las convenciones del proyecto (actualmente GDScript con comentarios mixtos/español).
